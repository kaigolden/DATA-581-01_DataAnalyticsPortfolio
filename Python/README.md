# Project: Penguins Exploratory Data Analysis (EDA)

## Overview
This project was an EDA of the Palmer Penquins dataset. Working in Python, I framed my analysis for a non-technical audience, created and interpreted a set of visualizations, and compiled a technical report covering my analysis introduction through my analysis methodology and a discussion of the results.

### What is included
- An audience-focused executive summary that translates the analysis purpose and findings into plain language.
- Several data visualizations, each with its interpretation.
- A reflection covering the results and ethics of the analysis.
- A discussion of further questions raised by the data and findings.

## Objective
Explore whether body mass differs across the three penguin species in the penguins dataset. 
## Dataset
[github: penguins.csv](https://github.com/mwaskom/seaborn-data/blob/master/penguins.csv) 
## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
## Findings
Gentoo penguins were considerably heavier than Adelie and Chinstrap penguins.
