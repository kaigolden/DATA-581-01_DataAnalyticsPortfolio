# PowerBI Projects

## Objective
Use HR Employees dataset to create various EDA visualizations in the Power BI app.

### What is included
- A .pbix file copy of the dashboard.
- HR Employees .xlsx dataset
- A link to a visualizations dashboard on the Power BI app.
  1. ** Note: an account is needed to view the dashboard.

## Dataset
HR_Employees.xlsx

## Tools
- [Power BI App](https://app.powerbi.com/)
- Power BI App account
- Application to open .pbix file
